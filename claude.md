# Allergen Alert

## What we’re building
A tool to scam menus and alert common allergens that are present. 

## Course Concepts
Using AI to replace cognitive load 

## Design Preferences
Plain language, no jargon 
Minimalistic aesthetic 
Yellow and white palette
HDML page 

## Rules
Never give information without checking online with a reputable source
Always state citation
Always cross check between different reputable websites

## Prompts:
- After sprint 1: Can we include more common allergens? Then add a tab function to sort each food menu item into different allergens, so the user doesn't have to scroll through manually over each food item.
- After sprint 2: Can we add a red disclaimer at the top of 
the page warning users to always double check with  the restaurant? and that this output is just a likely suggestion.
