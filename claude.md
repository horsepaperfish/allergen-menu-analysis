# Allergen Alert

## What we’re building
A tool to scam menus and alert common allergens that are present. 

## Course Concepts
Using AI to replace cognitive load 

## Design Preferences
Plain language, no jargon 
Minimalistic aesthetic 
Yellow and white palette
HDML page 

## Rules
Never give information without checking online with a reputable source
Always state citation
Always cross check between different reputable websites
